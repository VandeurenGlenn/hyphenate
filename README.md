# @vandeurenglenn/hyphenate

> MyEl -> my-el

## install
```sh
npm i @vandeurenglenn/hyphenate
```

## usage
```js
import hyphenate from '@vandeurenglenn/hyphenate'

hyphenate('MyEl')
```
